# Service-mfa

A service for adding multi-factor authentication on top of Pryv.io login calls.

Prerequisites: Node v8+, Yarn v1+

## How to?

| Task                              | Command                        |
| --------------------------------- | ------------------------------ |
| Setup                             | `yarn install`                 |
| Run API server                    | `yarn start`                   |
| Run Tests                         | `yarn test`                    |
| Create Distribution               | `yarn release`                 |
